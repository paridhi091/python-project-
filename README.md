#  Length and Mass Unit Conversion

  ##   An overview of the project
  This simple Python-based unit converter allows users to convert *Length* and *Mass* values between commonly used units.  
  It is appropriate for novices and runs directly on the terminal or command line.

  The program displays a menu, asks the user for input, and accurately completes the necessary conversion.

  ---
 ## Features
  - Convert *Length units*:
  - Centimeter ↔ Meter  
  - Centimeter ↔ Inch  
  - Centimeter ↔ Kilometer  
  - Kilometer ↔ Centimeter  

- Convert *Mass units*:
  - Gram ↔ Kilogram  
  - Milligram ↔ Gram  
  - Gram ↔ Milligram  
  - Kilogram ↔ Pound  
  - Pound ↔ Kilogram  

- Simple text menu  
- Beginner-friendly  
- No external dependencies

2. Conversions of Test Lengths
For conversions like these, try using different values (normal, decimal, and zero):

Centimeters → Meters

3. Mass Conversion Tests
Put sample values in for:
Grams → Kilograms
Milligrams → Grams

4. Verify Invalid Inputs: Use letters, blank input, or incorrect menu selections.
